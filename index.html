How to use

Add the ngQuerysting.min.js file below you angular file and above your application

<script src="app/libs/angular/angular.min.js"></script>
<script src="app/libs/ngQuerystring.min.js"></script>

Reference the ngQueryString module in your application

angular.module('application', ['ngQueryString'])

Add the service to your controller

.controller('controller', ['$scope', 'ngQueryString', function ($scope, ngQueryString) angular.module('test', ['ngQueryString'])

Use the ngQuerysting as you wish

e.g.

ngQueryString.setQueryString(name, value);

$scope.getQueryString.results = ngQueryString.getQueryString("name");

for further information on how to use please review the demo code here GitHub
